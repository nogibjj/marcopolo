# marco polo command line
step 1: create new project
`cargo init --name marcopolo`

step 2: check out structure by using `tree`
```bash
.
├── Cargo.toml
├── LICENSE
├── Makefile
├── README.md
└── src
    └── main.rs
```

step 3: `make run`

step 4: run the target `./target/debug/marcopolo`

step 5: `touch src/lib.rs`
